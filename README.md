# Greta V2 – AI Robotics Rover

Greta V2 is a self-engineered robotics platform combining embedded systems, AI modules and mobile control. This project focuses on building an intelligent rover capable of autonomous behaviour and human interaction.

## Key Features

• Arduino Uno motor control platform
• ESP32-S3 AI processing module
• Bluetooth mobile control
• Voice activation capability
• Obstacle avoidance system
• Modular architecture for upgrades

## Architecture

ARDUINO → Low level motor control
ESP32 → AI + communication
Dashboard → Control interface
Docs → Technical documentation

## Hardware Stack

• Arduino Uno
• ESP32-S3
• L298N Motor Driver
• Ultrasonic sensor
• DC geared motors
• Breadboard prototyping
• Portable power supply

## Development Goals

Greta is designed as a learning + engineering portfolio project.
